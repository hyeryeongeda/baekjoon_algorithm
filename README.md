# 📘 Baekjoon Algorithm Study

백준 온라인 저지(BOJ) 문제를 단계적으로 풀면서 기록한 저장소입니다.
주로 **Python**을 사용하며, 기초적인 구현부터 슬라이딩 윈도우, 문자열 처리 등 다양한 알고리즘을 연습합니다.

---

## ✅ 풀이한 문제 목록

## 🚀 앞으로의 목표

- 단계별 풀이([https://www.acmicpc.net/step](https://www.acmicpc.net/step)) 따라가며 기초를 다진다.
- 누적합, 투포인터, BFS/DFS, DP 등 기본 알고리즘 유형 정복.
- 꾸준히 기록해서 성장 과정을 확인한다.
